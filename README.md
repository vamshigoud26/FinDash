# Welcome to your Finance Insights Hub project

Finance Insights Hub is a React + Vite app styled with Tailwind CSS and powered by modern tools.

## 🚀 Setup

1. Install dependencies

```bash
npm install
```

2. Run in development mode (auto-opens browser):

```bash
npm run dev
```

- defaults: `--host 0.0.0.0 --port 5173 --open`
- it auto-falls back to next available port if 5173/5174/5175 are in use

3. Build production bundle:

```bash
npm run build
```

4. Preview the production build:

```bash
npm run preview
```

5. Run unit tests:

```bash
npm test
```

## 🧩 Notes

- App name: `finance-insights-hub`
- TypeScript + React + Tailwind + Vitest + Playwright
- For browser theme support, this project uses `next-themes`.
